# SaaS 서비스 비교 분석

## 평소 이용하는 서비스 목록

- Disney+
- Notion
- GitHub
- Photoshop
- Steam

## SaaS 인 서비스

- Disney+ : 설치 없이 웹에서 스트리밍을 이용하며, 구독 모델을 가지고 있음.
- Notion : 웹에서 문서 작성 및 데이터베이스 관리
- GitHub : 클라우드 환경에서 소스 코드 관리 및 배포

## SaaS 가 아닌 서비스 및 이유

- Photoshop : 소프트웨어를 직접 설치 후 실행해야함
- Steam : 소프트웨어를 직접 설치 후 실행해야함

## Netflix 의 SaaS 특징

1. 웹 브라우저로 접속
2. 설치 없이 웹에서 바로 콘텐츠 스트리밍 제공
3. 유료 구독 모델을 가지고 있음
